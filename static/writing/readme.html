Cedar

A lightweight static site generator built-in go with tree-sitter code-block highlighting.

Installation

Go install:

go install codeberg.org/pdewey/cedar@latest

From Source:

git clone https://codeberg.org/pdewey/cedar.git
cd cedar
go run main.go
